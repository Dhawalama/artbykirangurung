# Art by Kiran Gurung

A bilingual editorial portfolio website for Kiran Gurung, built with Astro and Tailwind CSS and deployed on GitHub Pages.

Live site:

```text
https://dhawalama.github.io/artbykirangurung/
```

---

## Stack

- Astro 5
- Tailwind CSS
- GitHub Pages
- GitHub Actions
- Static bilingual routing (`/en/` and `/ne/`)

---

## Routes

English:

```text
/en/
```

Nepali:

```text
/ne/
```
