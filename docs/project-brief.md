# Project brief

## Direction

Build a bilingual static artist portfolio for Kiran Gurung using the editorial standalone design as the visual source of truth.

## Pages / sections

- Home
- Selected works / Portraits
- Murals
- Digital Art
- Calendars
- Commissions
- Classes
- Shop

## Hosting

- GitHub Pages
- GitHub Actions CI/CD
- Static output only

## Commerce and forms

- Calendar/products link to Etsy for v1
- Commission/class enquiries use an external form provider
- No server-side Stripe/Resend in v1

## Visual principles

- Quiet editorial gallery
- Large serif hero type
- Generous whitespace
- Artwork-led sections
- Minimal product language
- Red accent used sparingly
- Bilingual English/Nepali support
